# 🏥 Сайт медицинской диагностики

> 💡 Современный сайт для клиники диагностики на Django + Bootstrap с поддержкой Docker и админкой

---

## 📑 Описание проекта

Это веб-сайт для компании медицинской диагностики, разработанный на основе фреймворка **Django** с использованием **Bootstrap 5** для адаптивного дизайна и контейнеризованный с помощью **Docker**. Проект предоставляет:

- Пользователям:
  - Возможность ознакомиться с услугами.
  - Записаться на приём.
  - Просматривать историю обращений.

- Администраторам:
  - Управление контентом и данными через удобную панель администрирования Django.

## ▶️ Как запустить проект локально

### 🔧 Предварительные требования:
- [Docker](https://www.docker.com/ )
- [Docker Compose](https://docs.docker.com/compose/install/ )

### 📦 Шаги установки:

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/PavelPugachenko/Diploma_project_python.git 
   cd Diploma_project_python
2. **Создайте .env файл:**
   ```bash
   cp .env.example .env
3. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
4. **Запустите проект через Docker Compose:**
   ```bash
   docker-compose up -d --build
5. **Выполните миграции:**
   ```bash
   docker-compose exec web python manage.py migrate
6. **Создайте суперпользователя:**
   ```bash
   docker-compose exec web python manage.py createsuperuser
7. **Откройте сайт в браузере:**
   ```bash
   http://localhost:8000
8. **Доступ к админке:**
   ```bash
   http://localhost:8000/admin
